# Simple Prints & RVGPT Setup

## Files Created:

### rvgpt.py - Standalone AI Chatbot
- Copy anywhere and run
- Text + Voice + Image support
- Syncs with product tracker
- Dark mode interface

### server.py - Product AI (old)
- Used for product details in Simple Prints
- Smaller, faster responses

## Quick Start

### RVGPT (Chatbot):
```bash
python3 rvgpt.py
```
Opens at http://localhost:8080

### Simple Prints Server:
```bash
python3 server.py
```
Runs at http://localhost:8000

## Get Free Gemini API Key

1. Go to: https://aistudio.google.com/apikey
2. Sign in with Google
3. Click "Create API Key"
4. Copy the key
